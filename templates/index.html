Welcome

Agriculture is the backbone of society, providing essential necessities for living all over the world, and will continue to be a crucial service far into the future. When it comes to tractor passes over a large field, any overlap in tractor pathing results in wasted resources. Seeding, pesticide distribution, herbicide distribution, and harvesting especially can result in massive costs incurred due to overlapping or underlapping the areas that tractor’s implements pass over.

While the technology does exist to provide positioning solutions that result in high precision positioning, it is very expensive to implement. Our project explores the possibility of using cheaper equipment to achieve acceptable pass to pass accuracy in agriculture. Our project dealt with the creation of a software package capable of quantifying the various relative accuracy parameters important to broad acre precision agriculture as well as the visualization and analysis of those parameters.

Our Solution

Our project’s software package consists of 2 different suites. The first is error quantification and detection software that calculates and flags drift errors, pass to pass variation and position jumps in precision agriculture data sets. The second software suite simulates tractor passes over a user selected area and feeds the positional data back to the error detection software for analysis, Keeping these two suites separate allows them to be used to test the functionality and adaptability of each other. Any assumptions coded in the simulation suite are directly derived from observations made of our two data sets described here.

The data collection was done at the parking lot intersecting CCIT and the ENGG dorms. Points were collected using an R10 RTK sensor at the edges of 13 parking lot line segments. This was done to both calibrate the RTK for our purposes and to provide simulation data for a vehicle’s path. A second set of static GNSS data was provided by our advisor Greg Roesler from one of the static receiver arrays run by NovAtel.
